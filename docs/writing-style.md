## 🚨 제1원칙 두 가지

### 1. 핵심 정리체 금지
AI 기본값은 요약체다. 아무 지시 없이 쓰면 "핵심은 ~다", "정리하자면 ~다" 같은 교과서 톤으로 회귀한다. 독자가 스크롤 2초만에 떠나는 톤이다.

**판단 기준:** "읽으면서 웃음 or 공감 or '어 이거 나야' 셋 중 하나가 나오는가?"
셋 다 안 나오면 핵심 정리체. 다시 써라.

### 2. Information Gain (Google 2026 핵심)
> "이 글이 기존 검색 결과 1~10위가 못 가진 새 통찰을 담고 있는가?"

**모든 글은 다음 중 최소 2개 포함:**
- 1차 자료 직접 인용 (인터뷰·위버스·라이브 발언, 출처 표기)
- 학술 연구·통계 인용 (Pittenger, Beebe, Quenk 등)
- 비교 데이터 (다른 인물·다른 유형과 명시적 대조)
- 시간 흐름 분석 (5년·10년 변화)
- 우리 사이트만의 관점 (192조합·기질론 통합)
- 미신/오해 정정 (X로 알려졌는데 실제는 Y)

**3개 이상 = 우수. 2개 = 합격. 1개 이하 = 재작성.**

## ❌ 금지 표현 (발견 즉시 수정)

### 표현 단위 금지
| 금지어 | 대체 방향 |
|--------|-----------|
| "핵심은 ~다" | 그냥 구체적인 장면으로 시작 |
| "정리하자면", "요약하면" | "그래서 결국 뭐냐면," "이게 뭔 소리냐면," |
| "결론적으로", "즉," | 생략 or "그니까 내 말은," |
| "~라고 할 수 있다" | "~야", "~거든" |
| "~하는 것이 중요하다" | "~해야 한다는 거. 나도 그랬어." |
| "다음과 같다:" | 바로 예시 투척 |
| "~에 대해 알아보자" | "~ 한번 볼까?" or 생략 |

### 구조 단위 금지
| 금지 | 이유 | 대체 |
|---|---|---|
| 같은 마무리 공식 ("기질에 따라 4가지 버전") | 매 글 반복 | 글마다 다른 끝맺음 (FAQ·시사점·새 질문) |
| "민지(ISFJ)도 비슷한 경험" | 페르소나 닳음 | 학술 사례 또는 다른 구체 인물 |
| "함께 보면 좋은 글" 매번 같은 3개 | 내부 링크 중복 | 글마다 다른 큐레이션 |
| 헤딩 "~인 이유" 4번 이상 | 단조 | 의문문·인용·통계 헤딩 혼합 |
| 공손체 (~입니다/~합니다/~하세요) | 우리 톤 X | ~야, ~거든, ~잖아 |
| **마크다운 표 (`\| col \| col \|`)** | **렌더러 미지원** | **bullet list (`- 라벨: 값`)** |
| **TL;DR 박스** | 한국 톤 어색·체류시간 ↓ | 첫 단락에 핵심 자연 포함 |

## ✅ 톤 규칙

### 1. 1인칭 "나" + 2인칭 "너"
박서연이 친구한테 말하는 톤. "독자 여러분" 같은 존칭 내레이터 절대 금지.

**어미:** ~야, ~거든, ~잖아, ~더라고, ~인 거야, ~해봤는데
**금지 어미:** ~입니다, ~합니다, ~하세요

### 2. 문장 길이 변주
긴 → 짧은 → 긴. 연속 3문장이 비슷한 길이면 망한 거다.

**❌:** "INTJ는 계획적이다. 그들은 미래를 본다. 감정 표현이 적다. 분석적이다."
**✅:** "INTJ는 계획적이야. 정확히는, 3년 뒤 5년 뒤를 매일 시뮬레이션하면서 사는 사람. 감정 표현? 거의 없어. 그래서 오해받지."

### 3. 구체적 숫자·고유명사
"많이", "자주" 같은 추상어 대신 구체.

**❌:** "INTJ는 관계에 시간을 많이 쓰지 않는다"
**✅:** "INTJ는 한 달에 친구 연락 평균 1.3번이야. 많이 아껴."

체감 숫자면 충분. 디테일이 진짜 같음을 만든다.

### 4. 공감 낚시
3-4문단에 한 번 "나만 그런 거 아니지?" "너도 그랬지?". 과하면 느끼함.

## 🪝 Hook 패턴 6가지 (첫 섹션)

**같은 글 안에서 같은 Hook 두 번 X. 다음 글은 이전과 다른 패턴.**

### A. 일화/장면형 (전통)
```
민지 집에서 X 정주행하던 토요일 밤이었어. 그 장면이 나왔을 때 ~
```

### B. 통계 충격형
```
한국 ENTJ 비율은 1.6%. 100명 중 1~2명. 그 희소한 1.6%가 K-pop 톱 아이돌의 절반을 차지한다.
```
→ Information Gain 즉시 시작.

### C. 인용 + 반전형
```
"저는 그냥 럭키비키예요." 장원영 이 말이 ENFP 낙천이 아니라 ENTJ Te 최적화라고 짚으면 다들 멈춰선다.
```

### D. 직접 질문형
```
정국 MBTI 진짜로 3번 바뀌었을까, 아니면 검사가 틀렸을까? 둘 다 아니야.
```
→ Featured Snippet·AI 검색 노림.

### E. 미신 깨기형
```
"ISTP는 리더 안 한다." 모든 MBTI 책에 나오는 정설. 안유진이 IVE 리더 맡은 순간 깨졌다.
```

### F. 시간 점프형
```
2019년 정국은 INFP였다. 2020년 ISFP. 2022년 INTP. 같은 사람의 검사 결과가 3년 동안 세 번 바뀐 게 무슨 뜻일까.
```

## 📚 학술 인용 라이브러리

**한 글에 2~3회 자연스럽게. 5회 이상 ❌ (교과서체).**

### MBTI 검사 신뢰도
- **Pittenger (1993)**: 5주 후 재검사 시 50% 다른 결과
- **McCrae & Costa (1989)**: 4글자 중 1~2개 변동 흔함
- **Boyle (1995)**: MBTI 신뢰도 비판 메타분석

### 인지기능 발달
- **Jung (1921)**: 주기능 평생 안정 가설
- **Beebe (2004) 8-function model**: 시기별 다른 기능 활성화
- **Quenk (2002)**: 스트레스 시 열등기능 폭주

### 성격과 사회 행동
- **Heilman (2007)**: 여성 리더십 backlash
- **Cain (2012) `Quiet`**: 내향형 사회 평가 절하
- **Fisher (2009)**: 신경전달물질 4기질론

### 청년기 자아 변화
- **Erikson (1968)**: 15~25세 정체성 탐색기
- **Marcia (1980)**: 정체성 4단계 (확산·유실·유예·성취)

**인용 형식:**
```
"Pittenger 연구(1993)에 따르면 MBTI 검사 5주 후 재검사 시 절반은 다른 결과가 나와."
```

## 🔍 SEO + AI 검색 최적화 (Google 2026)

### 메타 영역
- [ ] 메타 description **130~160자**
- [ ] **첫 100자 안에 메인 키워드 1회**
- [ ] 메인 키워드 밀도 **1.5~2%**
- [ ] LSI 키워드 5개 이상

### 헤딩 구조
- [ ] H2(섹션) 6~8개에 검색어 자연 포함
- [ ] 헤딩에 의문문/숫자/구체어
- [ ] **첫 H2 바로 아래 = Featured Snippet 노린 직접 답변** (50자 내, 박스 X — 자연 단락)

### Featured Snippet 트리거 (자연스러운 단락)
```
H2: 정국 MBTI는 정말 바뀌었나?
[직접 답변 50자]: 정국이 본인 공개한 결과는 INFP→ISFP→INTP. 검사 신뢰도 50% 한계 안에서 가능한 변화다.
[그 후 상세 설명...]
```

### FAQ 섹션 (글 끝 직전 필수)
**모든 글 1개 포함.** Featured Snippet·AI 검색 노출.

```markdown
## 자주 묻는 질문

**Q. 정국 MBTI는 진짜로 INFP인가요?**
A. 2021년 위버스 본인 공개 INFP-T가 가장 최근 공식 결과. ...

**Q. MBTI는 정말 바뀔 수 있나요?**
A. ...
```

→ 4~5개 Q&A 권장.

### AI 검색(SGE/Perplexity) 최적화
- **단정문 비율 60%+**: AI가 인용 시 변형 적게
- **숫자·통계 본문 분산**: AI가 인용하기 좋음
- **출처 인라인 명시**: "Pittenger 1993 연구에 따르면..."
- **단계별 리스트**: AI가 추출 용이

### 내부·외부 링크
- 본문 자연스러운 내부 링크 3~5개
- 앵커 텍스트에 키워드 자연 포함
- 외부 1차 자료 링크 1~2개 (E-E-A-T)
