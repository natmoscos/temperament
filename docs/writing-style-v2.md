## 📐 제1원칙: Information Gain (Google 2026 핵심)

Google March 2026 Core Update의 가장 중요한 시그널:

> **"이 글이 기존 검색 결과 1~10위가 못 가진 새 통찰을 담고 있는가?"**

**v2 모든 글은 다음 중 최소 2개를 포함해야 한다:**

- [ ] 1차 자료 직접 인용 (인터뷰·위버스·라이브 발언, 출처 표기)
- [ ] 학술 연구·통계 인용 (Pittenger, Beebe, Quenk 등)
- [ ] 비교 데이터 (다른 인물·다른 유형과 명시적 대조)
- [ ] 시간 흐름 분석 (5년·10년 변화)
- [ ] 우리 사이트만의 관점 (192조합·기질론 통합)
- [ ] 미신/오해 정정 (X로 알려졌는데 실제는 Y)

**3개 이상 충족 = 우수. 2개 = 합격. 1개 이하 = 재작성.**

## 🪝 Hook 패턴 6가지 (첫 섹션 다양화)

**같은 글 안에서 같은 Hook 두 번 쓰지 마라. 다음 글은 이전과 다른 패턴 선택.**

### A. 일화/장면형 (v1 기본형)
```
민지 집에서 X 정주행하던 토요일 밤이었어. 그 장면이 나왔을 때 ~
```
→ 친근감, 공감, 첫 문단 몰입.
→ 단점: 반복되면 진부.

### B. 통계 충격형
```
한국에서 ENTJ 비율은 1.6%. 100명 중 1~2명. 그 희소한 1.6%가 K-pop 톱 아이돌의 절반을 차지한다.
```
→ 데이터로 호기심 폭발.
→ Information Gain 즉시 시작.

### C. 인용 + 반전형
```
"저는 그냥 럭키비키예요." 장원영의 이 한마디가 틀렸다고 말하면 사람들이 화낸다. 근데 ENFP 낙천이 아니라 ENTJ Te 최적화라고 정확히 짚으면 다들 멈춰선다.
```
→ 1차 자료 + 반전 = 강력한 SEO 시그널.

### D. 직접 질문형
```
정국 MBTI 진짜로 3번 바뀌었을까, 아니면 검사가 틀렸을까? 둘 다 아니야.
```
→ Featured Snippet 노림 + 독자 즉각 끌어당김.

### E. 미신 깨기형
```
"ISTP는 리더 안 한다." 모든 MBTI 책에 나오는 정설이야. 안유진이 IVE 리더 맡은 순간 그 정설이 깨졌어.
```
→ 기존 정보 vs 새 사실 대비.

### F. 시간 점프형
```
2019년 정국은 INFP였다. 2020년엔 ISFP. 2022년엔 INTP. 3년 동안 같은 사람의 검사 결과가 세 번 바뀌었다는 게 무슨 뜻일까.
```
→ 타임라인 = 자연스러운 서사 + 학술 인용 도입 좋음.

## 🏗 섹션 구조 다양화 매트릭스 (5섹션 탈피)

**5섹션 고정 ❌. 글 주제에 따라 4~7섹션 유동적.**

### 변화 서사 글 (정국·V·카리나)
```
1. Hook (시간 점프형 권장)
2. 변화 사실 정리 (타임라인 표·인용)
3. 학술 배경: MBTI 변화는 가능한가? (Pittenger·Beebe 인용)
4. 시기별 인지기능 분석 (3시기 = 3섹션 가능)
5. 메타 분석: 변화의 의미
6. FAQ (SEO 필수)
7. 자기 적용 + CTA (다양화 라이브러리에서)
```

### 단일 유형 분석 글 (김수현·박지훈·안유진)
```
1. Hook (일화/인용/미신 중 선택, 이전 글과 다른 패턴)
2. 본인 발언/공식 자료 1차 인용
3. 캐릭터/작품 사례 (구체적 장면)
4. 인지기능 메커니즘 (학술 배경 1~2문장)
5. 사회·문화 맥락 (한국 사회에서 이 유형이 받는 시선)
6. 자기 적용 + CTA
```

### 비교/대조 글
```
1. Hook (질문형/미신 깨기형)
2. 두 인물 핵심 차이 매트릭스
3. 각 인물 독자 사례
4. 공통점 (의외의 연결)
5. 독자에게 시사점
6. FAQ + CTA
```

**금지:** "같은 X여도 기질에 따라 다름" 헤딩 5번 연속. 같은 마무리 공식 반복 시 자동 재작성.

## 🔬 분석 깊이 4단계 모델

**v1까지 우리 글은 1~2단계에서 멈춤. v2는 최소 3단계까지 도달해야 한다.**

### 1단계: 표면 관찰 (필수)
```
"장원영은 럭키비키 ENTJ다."
```

### 2단계: 인지기능 해석 (필수)
```
"Te 주기능이 최적화 사고로 작동한다."
```

### 3단계: 심리학 메커니즘 (v2 추가 필수)
```
"Te 주기능자는 Quenk(2002) 분류에서 '결과 검증형' 의사결정을 한다.
일반인 대비 결정 속도 평균 2배, 후회율 30% 낮음."
```

### 4단계: 사회·문화 맥락 (v2 권장)
```
"한국 사회에서 ENTJ 여성은 '리더십 발휘 시 처벌(backlash)' 효과를 겪는다.
Heilman(2007) 연구: 같은 성과여도 ENTJ 남성 +호감, 여성 -호감.
장원영의 '럭키비키' 페르소나는 이 backlash를 우회하는 언어적 전략이다."
```

**최소 1번은 3~4단계까지 들어가라.** 그래야 글에 깊이가 생긴다.

## 📚 학술 인용 라이브러리 (자유롭게 활용)

**아래 연구·이론은 글에 자연스럽게 녹여 인용 가능.** 모두 검증된 1차/2차 자료.

### MBTI 검사 신뢰도
- **Pittenger (1993)**: MBTI 검사 5주 후 재검사 시 50% 다른 결과
- **McCrae & Costa (1989)**: 4글자 중 1~2개 변화 흔함
- **Boyle (1995)**: MBTI 신뢰도 비판 메타분석

### 인지기능 발달
- **Jung (1921) `Psychological Types`**: 주기능 평생 안정 가설
- **Beebe (2004) 8-function model**: 시기별 다른 기능 활성화
- **Quenk (2002) `Was That Really Me?`**: 스트레스 시 열등기능 폭주

### 성격과 사회 행동
- **Heilman (2007)**: 여성 리더십 backlash 효과
- **Cain (2012) `Quiet`**: 내향형 사회 평가 절하
- **Fisher (2009) `Why Him? Why Her?`**: 신경전달물질 기반 4기질론

### 청년기 자아 변화
- **Erikson (1968) `Identity`**: 15~25세 정체성 탐색기
- **Marcia (1980)**: 정체성 4단계 (확산·유실·유예·성취)

**인용 형식 (한국어 자연스럽게):**
```
"Pittenger 연구(1993)에 따르면 MBTI 검사 5주 후 재검사 시 절반은 다른 결과가 나와."
"Beebe의 8기능 모델은 나이에 따라 보조기능이 표면화될 수 있다고 본다."
```

**남용 금지:** 한 글에 학술 인용 5회 이상 = 교과서체. 2~3회가 최적.

## 🔍 SEO 최적화 체크리스트 (Google 2026)

### 메타 영역
- [ ] **메타 description 130~160자** (영어 글자 수 기준)
- [ ] **첫 100자 안에 메인 키워드 1회**
- [ ] **메인 키워드 밀도 1.5~2%** (자연스럽게)
- [ ] **LSI 키워드 5개 이상** (예: "정국 INFP" → "BTS 멤버 MBTI", "INFP 가수")
- [ ] **URL slug에 메인 키워드** (이미 컨벤션 따름)

### 헤딩 구조
- [ ] **H2(섹션) 6~8개에 검색어 자연 포함**
- [ ] **헤딩에 의문문/숫자/구체어 사용** ("왜", "3가지", "2025년")
- [ ] **첫 H2 바로 아래 = Featured Snippet 노린 직접 답변** (50자 내)

### Featured Snippet 트리거
```
H2: 정국 MBTI는 정말 바뀌었나?
[직접 답변 50자]: 정국이 본인 공개한 결과는 INFP→ISFP→INTP. 검사 신뢰도 50% 한계 안에서 가능한 변화다.
[그 후 상세 설명...]
```

### FAQ 섹션 (글 끝 직전 필수)
**v2부터 모든 글은 FAQ 섹션 1개 포함.** 구글이 구조화된 데이터로 인식.

```markdown
## 자주 묻는 질문

**Q. 정국 MBTI는 진짜로 INFP인가요?**
A. 2021년 위버스에서 본인이 공개한 INFP-T가 가장 최근 공식 결과예요. 단 본인이 "I와 P만 고정"이라고 발언한 점 고려하면 IxxP 체계 안에서 변동 가능성 있음.

**Q. MBTI는 정말 바뀔 수 있나요?**
A. ...

**Q. INFP인 BTS 멤버가 또 있나요?**
A. ...
```

→ 4~5개 Q&A 권장. Featured Snippet·People Also Ask 노출 가능성↑.

### 내부 링크
- [ ] **본문 중 자연스러운 내부 링크 3~5개** (관련 글)
- [ ] **앵커 텍스트는 키워드 자연 포함** ("정국 글 보기" ❌ → "정국 MBTI 변화 분석" ✅)
- [ ] **글 마지막 "함께 보면 좋은 글" 3개** (다양하게 — 매번 같은 3개 ❌)

### 외부 링크 (E-E-A-T 강화)
- [ ] **1차 자료 출처 외부 링크 1~2개** (위버스·인터뷰·연구 논문)
- [ ] `rel="nofollow"` 굳이 필요 X (정상 링크)

## 🎁 CTA 다양화 라이브러리 (8가지 변형)

**같은 CTA 문구 두 글 연속 사용 금지.**

### v1 기본형
```
[192가지 정밀 검사](/test)에서 네 MBTI + 기질 조합을 100문항 15분 무료로 확인할 수 있어.
```

### 변형 1: 결과 강조형
```
검사하고 나면 네가 ENTJ-담즙질인지 ENTJ-우울질인지가 갈려. 같은 ENTJ인데 완전 다른 사람이거든. [192가지 검사](/test) 15분이면 끝나.
```

### 변형 2: 비교 강조형
```
일반 MBTI 검사는 16칸이야. 우리는 192칸이야. 같은 INFP가 12명 다 다르게 살아. 어떤 INFP인지 [정밀 검사](/test)로 좁혀봐.
```

### 변형 3: 호기심 자극형
```
혹시 너 MBTI 결과가 자꾸 바뀐 적 있어? 그 흔들림의 답이 기질 조합에 있어. [192가지 검사](/test) 한번 해봐.
```

### 변형 4: 행동 유도형
```
점심시간에 한 번 해봐. 15분, 100문항. 결과지에 너만의 기질 좌표가 찍혀. → [검사 시작](/test)
```

### 변형 5: 차별화 강조형
```
MBTI는 "나는 ~다" 까지만 알려줘. 우리 검사는 "나는 어떻게 ~하는가"까지 풀어줘. [192가지 정밀 검사](/test) 무료.
```

### 변형 6: 친구 공유 유도형
```
민지(ISFJ)랑 이거 같이 했더니 "우리가 왜 자주 다투는지 이제 알겠다" 했어. 친구랑 같이 [검사](/test) 해봐.
```

### 변형 7: 학술 강조형
```
우리 검사는 Jung 인지기능 + 히포크라테스 4기질 통합 모델. 다른 검사 어디에도 없어. → [192가지](/test)
```

### 변형 8: 글 주제 직결형 (글마다 다르게)
```
[정국 글 끝]: 너도 IxxP면 검사 결과 자주 바뀐 적 있지? 변하는 표면 말고 안 변하는 뼈대를 보고 싶다면 → [기질 조합 검사](/test)
```

**규칙: 글 주제와 직결된 문구로 매번 새로 만든다. 위 8개는 참고 라이브러리.**

## 🎨 박서연 페르소나 사용 가이드 (v2 정제)

**v1에서는 박서연·민지·이준형이 모든 글에 등장했음. v2는 선택적 활용.**

- **글당 등장 횟수: 최대 2회** (페르소나 한 명 기준)
- **연예인 글에서는 박서연 = 덕질 시점만 가볍게.** 본문 분석은 학술·인지기능·1차 자료가 주 동력.
- **민지·이준형은 비교 사례 필요할 때만 1회 등장. 매 글 ❌.**
- **새 페르소나 도입 가능:**
  - **박지원 (28세, 마케터, ENTJ)** — 비즈니스·커리어 글에 적합
  - **이서영 (32세, 프리랜서, INFP)** — 창작·예술 글에 적합
  - **김도윤 (29세, 개발자, INTP)** — 분석·시스템 글에 적합

→ 페르소나 다양화 = 글 다양화의 핵심 축 중 하나.

## 변우석을 닮은 사람이 한국에 5만 명 — 어디 있을까?

ENFJ 비율은 한국 인구 1.7%. 5,000만 명 기준 약 85만 명. 근데 변우석처럼

### 표면 ENFJ vs 깊은 ENFJ — Ni 보조의 발달 정도가 갈라

ENFJ 주기능 Fe(외향 감정)는 누구나 비슷하게 작동한다. 차이는 보조기능 Ni
(내향 직관)에 있다. Ni가 미성숙한 ENFJ는 단발 친절형(이번 만남만 따뜻).
Ni가 성숙한 ENFJ는 장기 관계 설계형(5년 후를 그리며 지금 행동).

변우석의 우통 컨셉은 후자의 정수다. 같은 팬을 5년째 같은 애칭으로 부르며
배경·취향까지 기억하는 패턴 = Ni 보조가 4단계까지 발달한 ENFJ.
